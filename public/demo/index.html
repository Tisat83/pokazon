Интерактивное демо

Слева — окно оператора, справа — окно клиента. Повторите шаги ниже, чтобы увидеть управление в действии.

  1. В правом окне нажмите «Подключить» — появится код.
  2. Нажмите на код, чтобы скопировать его.
  3. В левом окне вставьте код в поле и нажмите «Войти».
Далее двигайте мышь в левом окне (оператор) и наблюдайте действия в правом (клиент). Управление ограничено только этой страницей.

Окно оператора

Если панель закрыта — откройте её комбинацией Ctrl+Alt+O.

Окно клиента

Нажмите «Подключить» для получения кода. Код можно кликнуть — он скопируется.
Для реального теста на двух устройствах откройте отдельно: клиент — /#help, оператор — /#operator.
