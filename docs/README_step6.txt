
OKO – шаг 6: улучшенное управление
- Click по координатам: элемент выбирается на стороне клиента через elementFromPoint(x,y).
- Ввод с клавиатуры: оператор печатает у себя, а символы применяются к активному полю ввода у клиента.
- Сообщение на странице оператора при завершении сессии.

Проверка:
1) npm run dev
2) Откройте /demo и /operator. Создайте сессию, подключитесь.
3) Кликните в поле "Количество" у клиента (через операторскую вкладку — кликом в координату поля): сначала наведите курсор над соответствующей позицией, затем кликните. Фокус должен попасть в поле у клиента.
4) Печатайте цифры на клавиатуре во вкладке оператора — значение у клиента должно меняться. Backspace работает.
5) Нажмите "Стоп" у клиента — оператор увидит "Сессия завершена хостом."

Ограничение: координаты интерпретируются в пределах viewport (без учёта масштаба/скролла внутри вложенных контейнеров).
